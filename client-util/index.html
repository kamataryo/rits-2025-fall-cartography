Vote Client Library Test

このページは投票クライアントライブラリのテスト用です。

1. 自由テキスト投票

自由にテキストを入力して投票します。freetext="on" 属性を使用。

2. 選択肢投票（ラジオボタン）

選択肢から1つを選んで投票します。choices="選択肢1,選択肢2,..." 属性を使用。

3. プログラミング言語投票

好きなプログラミング言語を選んで投票します。

4. 混合投票（ラジオボタン + 自由テキスト）

選択肢から選ぶか、自由にテキストを入力できます。choices と freetext="on" の両方を使用。

5. 授業評価投票

授業の理解度について投票します。

6. 属性なし（デフォルト）

choices も freetext も指定しない場合（submitボタンのみ表示）

7. フリーフローモード

8. リアクションコンポーネント（カスタムEmoji）

emojis 属性でカスタムEmojiを指定。複数のEmojiから選択可能。

9. カスタムスタイル - ダークテーマ

<style> 要素を使用してダークテーマを適用。背景色、文字色、ボタンスタイルをカスタマイズ。

10. カスタムスタイル - グラデーションテーマ

美しいグラデーション背景とガラスモーフィズム効果を適用。

11. カスタムスタイル - ミニマルテーマ

シンプルで洗練されたミニマルデザイン。余計な装飾を排除。

12. カスタムスタイル - 企業ブランドテーマ

企業のブランドカラーを使用したプロフェッショナルなデザイン。

13. カスタムスタイル - フリーフロー + カスタムテーマ

フリーフローモードにカスタムスタイルを適用。アイテムの表示をカスタマイズ。

14. カスタムスタイル - アニメーション効果

CSS アニメーションとトランジション効果を活用したインタラクティブなデザイン。

15. カスタムスタイル - レスポンシブデザイン

モバイルデバイスに最適化されたレスポンシブデザイン。画面サイズに応じてレイアウトが変化。

使用方法

  1. WebSocketサーバーがデプロイされていることを確認
  2. ブラウザの開発者ツールでコンソールを開く
  3. 各フォームで投票を試す
  4. リアルタイムでチャートが更新されることを確認
  5. 複数のブラウザタブで開いて、ブロードキャスト機能を確認
  6. リアクションボタンをクリック・ホバー・長押しして動作確認
  7. 他のデバイスからリアクションを送信して、飛行アニメーションを確認

カスタムスタイル機能

セクション9-15では、新しく追加されたカスタムスタイル機能のサンプルを確認できます：

  • ダークテーマ: 暗い背景色と明るい文字色を使用
  • グラデーションテーマ: 美しいグラデーション背景とガラスモーフィズム効果
  • ミニマルテーマ: シンプルで洗練されたデザイン
  • 企業ブランドテーマ: プロフェッショナルな企業向けデザイン
  • フリーフロー + カスタム: フリーフローモードにカスタムスタイルを適用
  • アニメーション効果: CSS アニメーションとトランジション効果
  • レスポンシブデザイン: モバイルデバイス対応のレスポンシブレイアウト

カスタマイズ可能な要素

<style> 要素内で以下のクラスをカスタマイズできます：

  • .vote-container - 全体のコンテナ
  • .vote-contents - フォームとチャートの親要素
  • .vote-form - フォーム全体
  • .form-group - フォームグループ
  • .radio-group, .radio-item - ラジオボタン関連
  • button - 送信ボタン
  • input[type="text"], input[type="radio"] - 入力要素
  • label - ラベル
  • .chart-container - チャートコンテナ
  • .freeflow-container, .freeflow-item - フリーフロー関連
  • .connection-status - 接続状態表示

使用例

<vote-form vote-key="my-vote" choices="選択肢1,選択肢2">
  <style>
    .vote-container {
      background: linear-gradient(45deg, #ff6b6b, #4ecdc4);
      border-radius: 20px;
      color: white;
    }

    button {
      background: rgba(255, 255, 255, 0.2);
      border: 2px solid white;
      border-radius: 25px;
    }
  </style>
</vote-form>
