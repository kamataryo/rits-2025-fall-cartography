Vote Client Library Test

このページは投票クライアントライブラリのテスト用です。

1. 自由テキスト投票

自由にテキストを入力して投票します。freetext="on" 属性を使用。

2. 選択肢投票（ラジオボタン）

選択肢から1つを選んで投票します。choices="選択肢1,選択肢2,..." 属性を使用。

3. プログラミング言語投票

好きなプログラミング言語を選んで投票します。

4. 混合投票（ラジオボタン + 自由テキスト）

選択肢から選ぶか、自由にテキストを入力できます。choices と freetext="on" の両方を使用。

5. 授業評価投票

授業の理解度について投票します。

6. 属性なし（デフォルト）

choices も freetext も指定しない場合（submitボタンのみ表示）

7. リアクションコンポーネント（デフォルト）

右下に固定表示される👍ボタン。マウスオーバーまたは長押しで展開、クリックでリアクション送信。

8. リアクションコンポーネント（カスタムEmoji）

emojis 属性でカスタムEmojiを指定。複数のEmojiから選択可能。

9. リアクションコンポーネント（授業用）

授業で使いやすいEmojiセット。理解度や感情を表現。

使用方法

  1. WebSocketサーバーがデプロイされていることを確認
  2. ブラウザの開発者ツールでコンソールを開く
  3. 各フォームで投票を試す
  4. リアルタイムでチャートが更新されることを確認
  5. 複数のブラウザタブで開いて、ブロードキャスト機能を確認
  6. リアクションボタンをクリック・ホバー・長押しして動作確認
  7. 他のデバイスからリアクションを送信して、飛行アニメーションを確認
